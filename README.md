galaxygen
=========

Randomly generate an EVE Online-esque galaxy using Python

Dependencies
-------------
Python 2.7

numpy, scipy, networkx (for graph algorithms)

PyQt4 (to generate the galaxy image)

Usage
-----
Navigate to the folder in a terminal and run "python main.py"
